🌊 AlphaEarth 核心场景升级方案：江苏盐城红线审计 (V8.1 泛化与精度控制版)

一、 算法调优逻辑：从“过拟合”到“弹性泛化”

在前期 V8.0 版本的强共识交叉验证中，我们保障了样本的“绝对纯净”，但忽略了真实地理环境的“灰度渐变区”，导致模型面临局部过拟合与碎片化噪点的风险。

V8.1 版本引入了弹性控制机制 (Elastic Control)，旨在平衡“局部像素精度”与“宏观地貌泛化性”：

样本泛化 (Soft Margin)：释放过于严苛的掩膜条件，容许特征空间存在合理的“软边界”，覆盖真实海岸带的过渡态特征。

树的正则化 (Tree Regularization)：通过限制随机森林的树深与叶子节点容量，阻断算法死记硬背噪声，提升在未知海域的泛化能力。

形态学约束 (Morphological Smoothing)：在地学空间维度注入“平滑算子”，抹除非自然的椒盐碎斑，提升审计级制图的视觉连贯性和最终物理准确性。

二、 后端离线导出脚本 (ch5_rf_export.py) 泛化控制升级

本仓库已内置 V8.1 版本脚本：backend/ch5_rf_export.py（建议直接使用，不要将脚本内容粘贴到文档里，以免后续迭代导致文档与真实实现漂移）。

1) 配置资产路径（二选一）

- 推荐：CH5_RF_ASSET_ID 指定完整 Asset ID
- 备选：设置 GEE_USER_PATH 让脚本推导默认路径（/classifiers/ch5_coastline_rf_v1）

2) 执行导出

```bash
export CH5_RF_ASSET_ID=users/<your_username>/aef_demo/classifiers/ch5_coastline_rf_v1
# 或：export GEE_USER_PATH=users/<your_username>/aef_demo

python backend/ch5_rf_export.py --check
python backend/ch5_rf_export.py --ensure
```

你会在 GEE Tasks 面板看到描述名：Export_Coastline_RF_V8_1（脚本只提交任务，不会等待完成）。

3) 关键参数对照（环境变量）

- CH5_RF_POINTS_PER_CLASS：每类采样点数（默认 3000）
- CH5_RF_SAMPLE_SCALE：采样分辨率（默认 30）
- CH5_RF_SEED：随机种子（默认 42）
- CH5_RF_TREES：RF 树数（默认 60）
- CH5_RF_MIN_LEAF_POP：最小叶子样本量（默认 10）
- CH5_RF_BAG_FRACTION：袋外采样比例（默认 0.6）

4) V8.1 软边界共识规则（概念摘要）

- 水体：JRC occurrence >= 80 且 ESA 为水
- 滩涂：5 < occurrence < 80 且 ESA 非建成区
- 人工：ESA 建成区 且 occurrence <= 2
- 内陆：ESA 农田/林草 且 occurrence <= 2

注意：stratifiedSample 的 classBand 必须是整型，当前脚本已强制转换（如果你仍遇到报错，通常是运行了旧版本脚本）。


三、 后端服务渲染优化 (gee_service.py)

在后端的出图逻辑中，我们加入形态学滤波 (Morphological Smoothing)。这是地理空间分析中提升最终物理精度（准确性）的关键一步，它能把零散的预测结果“凝聚”成连贯的实体。

替换 gee_service.py 中的 ch5_coastline_audit 分支：

    elif ("ch5_coastline_audit" in mode_s) or ("海岸线" in mode_s) or ("红线审计" in mode_s):
        base_img = _select_embedding_bands(filtered_col.mosaic(), emb_bands)
        ASSET_ID = "projects/your-project/assets/classifiers/ch5_coastline_rf_v1" # 确认 Asset ID
        classifier = ee.Classifier.load(ASSET_ID)
        
        # 0=水, 1=自然滩涂, 2=人工围垦, 3=内陆林草
        img = base_img.classify(classifier)
        
        # 🛡️ 最终精度控制：形态学众数滤波 (Morphological Majority Filter)
        # 用 1.5 像素半径的核在图像上滑动，强行抹平孤立的“椒盐噪点”，让防波堤和滩涂的边界变得连贯、清晰，提升宏观准确度。
        img = img.focal_mode(radius=1.5, kernelType='circle', iterations=1)
        
        # 空间地理围栏 (Coastal Geofence)
        coastal_fence = ee.Geometry.Polygon([[
            [120.30, 34.00], 
            [121.50, 34.00], 
            [121.80, 32.50], 
            [120.60, 32.50], 
        ]])
        
        # 裁剪地理围栏
        img = img.clip(coastal_fence)
        
        # 语义透明化
        img = img.updateMask(img.neq(3).And(img.neq(0)))
        
        vis = {
            "min": 0, 
            "max": 3, 
            "palette": [

                "E23D28",  # 2: 人工围填海/高强度开发 (红)
                "000000"   # 3: 内陆背景 (已透明)
            ]
        }
        suffix = "ch5_audit_v8_1_generalized"
        return img, vis, suffix
