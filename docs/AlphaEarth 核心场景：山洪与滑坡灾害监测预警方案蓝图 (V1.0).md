一、 业务痛点与传统技术瓶颈

在西南山区、西南横断山脉等区域，山洪与滑坡往往伴随发生（暴雨诱发）。传统的监测手段面临巨大挑战：

光学卫星的“云遮挡诅咒”：地质灾害多发于汛期和暴雨后，传统光学卫星（如 Sentinel-2）几乎100%被厚云遮挡，无法做到极速响应。

物理模型的“参数爆炸”：传统水文泥石流模型需要输入极高精度的土壤类型、坡度、降雨量等数十个参数，在广袤的无人区根本无法获取。

传统深度学习的“样本荒”：滑坡形态千奇百怪，且属于“长尾小概率事件”。用传统 CNN 网络去学习，根本收集不到足够的多边形标注样本，模型泛化力极差。

二、 AEF 范式的“降维打击”优势

结合 V8.1 白皮书的理念，AEF 基础大模型在灾害场景下具备以下革命性优势：

“穿云透雾”的语义连续性
AEF 大模型在预训练时融合了多模态数据（包括不受云层影响的 SAR 微波雷达数据）。因此，即便在汛期多云天气，AEF 的 16 维特征空间（特别是 A02 介电常数/水分维度、A00 硬度纹理维度）依然能稳定输出地表的物理骨架。

天然的地质拓扑理解能力
滑坡的物理本质是“原本连续的植被/山体结构发生了撕裂”。AEF 的隐空间特征天然内化了空间拓扑结构。当山体滑坡发生时，特征空间中的 A01（三维冠层生物量）会瞬间断崖式下跌，同时 A00（裸露岩石/土壤硬度）和局部纹理几何特征会发生剧烈畸变。

极小样本的“灾后极速定损”
如同海岸线审计，我们不需要人工标注滑坡体。只需利用极少量的历史已知滑坡点，甚至使用零样本/无监督的“跨期特征空间欧氏距离”，就能瞬间标红整座山脉的破裂带。

三、 系统化解决方案：多源共识与时空差分

针对山洪与滑坡，我们设计一套类似 V8.1 的“多源共识漏斗”与 V9.0 的“时空状态差分”架构：

1. 数据基座引入“灾害地质金标准” (Multi-Source Consensus)

如同海岸线依赖 ESA 和 JRC，山区灾害我们也需要引入国际金标准作为空间约束围栏 (Geofence)：

地形约束 (Copernicus DEM 30m)：提取 Slope (坡度) > 15° 的区域。滑坡绝不可能发生在平原，直接掩膜掉平原区，极大减少算力消耗和误判。

汇水区约束 (HydroBASINS)：提取山谷的集水线（Flow Accumulation），只在这些经脉上寻找山洪爆发的痕迹。

降水触发器 (GPM 全球降水)：作为时间切片的触发器，提取暴雨发生前后的 AEF 特征切片进行对比。

2. 算法机理：隐空间剧变检测 (Latent Space Mutagenesis)

这是一种无需重新训练复杂模型的“差分突变检测”策略：

滑坡破裂带提取 (Landslide)：

算子：计算暴雨前（Pre）和暴雨后（Post）的 AEF 16维向量的 欧氏距离 ($L_2$ Norm)。

过滤：叠加 A01 (植被) 显著下降 且 Slope > 15° 的条件。

结论：在陡坡上发生的剧烈结构突变且伴随植被消失，几乎 100% 是滑坡或泥石流！

山洪淹没区提取 (Flash Flood)：

算子：聚焦 A02（介电常数/水分）维度的时序差分（$\Delta A02$）。

过滤：叠加 DEM 山谷汇水区 条件。

结论：在历史非水域的山谷低洼处，水分特征突增，即为山洪淹没和河道溃决区。

        # 2. 计算 16 维空间的全局突变程度 (欧氏距离)
        distance = emb_pre.subtract(emb_post).pow(2).reduce(ee.Reducer.sum()).sqrt()
        
        # 3. 提取关键物理维度的异动
        delta_A01 = emb_post.select('A01').subtract(emb_pre.select('A01')) # 植被增减
        delta_A02 = emb_post.select('A02').subtract(emb_pre.select('A02')) # 水分增减
        
        # 4. 引入地质地形金标准 (DEM 坡度)
        dem = ee.Image("COPERNICUS/DEM/GLO30")
        slope = ee.Terrain.slope(dem)
        
        # 5. 🎯 靶向诊断逻辑：
        
        # [滑坡/泥石流诊断]：高陡坡 (>15°) + 全局突变剧烈 (>0.2) + 植被骤降 (< -0.1)
        is_landslide = slope.gt(15).And(distance.gt(0.2)).And(delta_A01.lt(-0.1))
        
        # [山洪/淹没区诊断]：低洼平缓地带 (<15°) + 水分特征骤增 (> 0.15)
        is_flood = slope.lte(15).And(delta_A02.gt(0.15))

        out = out.where(is_flood, 1)     # 1: 山洪 (蓝色)
